Package prifi-sda-protocol contains the SDA protocol that transmits prifi-lib messages through the network. As of now it only contains a dummy example protocol for demonstration purpose.

    type MyMsg ¶ 

    type MyMsg struct {
    Text string
}

    MyMsg contains a single message string

    type MyMsgStruct ¶ 

    type MyMsgStruct struct {
    *sda.TreeNode
    MyMsg
}

    MyMsgStruct is used by the SDA to transmit messages of type MyMsg. The message handler functions must take a struct containing a network registered message struct (here MyMsg) and a *sda.TreeNode which will contain the TreeNode that sent the message.

    type MyProtocol ¶ 

    type MyProtocol struct {
    *sda.TreeNodeInstance
}

    MyProtocol contains the protocol's state on one node and must implement the sda.ProtocolInstance interface (most of it is implemented by the contained TreeNodeInstance, we only have to implement Start())

    func (*MyProtocol) HandleMsg ¶ 

    func (p *MyProtocol) HandleMsg(msg MyMsgStruct) error

    HandleMsg handles a message of type MyMsg received by the SDA. It is automatically called by the SDA when a message of type MyMsg is received. See also MyMsgStruct comments.

    func (*MyProtocol) Start ¶ 

    func (p *MyProtocol) Start() error

    Start is called by the service when starting the protocol. It is part of the sda.Protocol interface.
